USB camera

Posted Dec 4, 2021 Updated May 12, 2023
By TheYOSH
1 min read

Information

Use any USB webcam that is supported by Linux.

Controls

As this is a Video4Linux device, you can use v4l2-ctl tools to control the focus, white balance and other options. This is not possible with the TerrariumPI software. A good website with more information and how to use it.

Webcam type: Still picture

Setup

In order to use the USB camera use the following settings:

Mandatory

Hardware
USB camera
Address
Enter the device path
Ex: /dev/video0

Other settings can be found at the webcam setup information
